# Telegram-chat-for-wordpress-pligin
Maestro Telegram Chat — это универсальный плагин для WordPress, который добавляет на сайт интерактивный чат-виджет с отправкой сообщений напрямую в Telegram. Пользователь может написать обращение через форму, а владелец сайта мгновенно получает заявку в личные сообщения или группу Telegram.  Плагин создан для тех, кто хочет получать заявки с сайта
